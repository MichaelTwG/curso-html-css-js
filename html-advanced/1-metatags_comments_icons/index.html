Comments

Un comentario es un codigo que el navegador no interpreta, es decir que no se muestra en la pagina web. Esta destinado a ser como una nota para los desarolladores. Se utiliza para describir que funciones tiene el codigo, como se utiliza, etc. Son importantes por dos razones la primera es que si volvemos a trabajar en ese codigo despues de mucho tiempo seguramente allamos olvidado que es lo que estabamos haciendo. El otro motivo es que si un desarollador que no seamos nosotros empieza a trabajar con nosotros o hereda el proyecto podra entender cuales eran tus intenciones con mas claridad.

En visual studio code podemos utilizar un atajo de teclado, lo primero que tenemos que hacer es seleccionar el codigo que queremos comentar y luego precionar control + k soltamos las teclas anteriores y luego pesionamos c. si queremos podemos descomentar, seleccionamos el codigo comentado y pulsacmos control + k y lugo la tecla u.

Favicons

Es el logo que aparece justo donde esta el nombre de la pestaña en el navegador. lo colocamos en la etiqueta head con la siguiente estructura:

<link rel="icon" src="ruta de la imagen" type="image/png" >

La imagen del logo tiene que ser de 64x64 pixeles o 32x32 pixeles

Metatags

La finalidad de describir datos, configuraciones sobre nuestra web, no son visibles para el usuario

Metatags

  • meta charset="UTF-8" - define el codigo de lenguaje "utf-8" permite todos los caracteres de todos los lenguajes
  • meta name="viewport" content="width=device-width, initial-scale=1.0"
  • meta name="desricption" content="descripcion de la pagina - cuando buscamos algo en los navegadores por lo general los resultados se componen del h1 y la description de la pagina (esa descripcion es la que define esta etiqueta)
  • meta name="keyword" content="metatags, comments html, favicons html" - se utiliza para definir las plabras clave que representan la tematica de la web. Importante para el posicionamiento de la web
  • meta name="author" content="Michael Golenia" - El nombre del autor de la página
  • meta name="robots" content="nofollow nosnippet noarchive noimageindex"
  • meta name="title" content="" - titulo de la web

Open Graph Metatags

  • meta property="OG:title" content=""
  • meta property="OG:description"
  • meta property="OG:image"
  • meta property="OG:site_name"
  • meta property="OG:url"
