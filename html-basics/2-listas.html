Listas Ordenadas y Listas Desordenadas

Existen dos tipos de listas las listas ordenadas (bullet) y las listas desordenadas (numeradas), generalmente suelen utilizarse las listas para crear los menús de navegacion.

Se componen de dos elementos el contenedor (ol u ul) y los elementos que componen la lista que son llamados li.

Listas ordenadas (etiqueta ol - ordered list)

  1. Encender la computadora
  2. Abrir el navegador
  3. Escribir la URL del sitio web

Listas desordenadas (etiqueta ul - unordered list)

  • HTML
  • CSS
  • JavaScript

Consejo: Las listas también se pueden anidar (poner una lista dentro de otra). Esto es muy útil para menús con submenús o para agrupar información jerárquica.
