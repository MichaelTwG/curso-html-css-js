Formulario de Pedido - Instrucciones

TAREA 1: Crea la etiqueta <form>.
- El formulario debe enviar los datos a una URL ficticia "/procesar-pedido".
- Debe usar el método "POST".

TAREA 2: Crea un <fieldset> para agrupar los datos de contacto.
- Añade una leyenda (<legend>) que diga "Datos de Contacto".

TAREA 3: Campo para el Nombre y Apellido.
- Usa una etiqueta <label> con el texto "Nombre y Apellido:".
- Usa un <input> de tipo "text".
- Haz que sea un campo obligatorio (required).
- Dale un name apropiado (ej: "nombre_cliente").
- Añade un placeholder con un ejemplo (ej: "Juan Pérez").
- No olvides vincular el <label> con el <input> usando for y id.


TAREA 4: Campo para el Teléfono.
- Usa una etiqueta <label> con el texto "Teléfono:".
- Usa un <input> de tipo "tel".
- Haz que sea obligatorio.
- Dale un name (ej: "telefono").
- Añade un placeholder (ej: "1122334455").
- Vincula el <label> con el <input>.


TAREA 5: Campo para la Dirección de Entrega.
- Usa una etiqueta <label> con el texto "Dirección de Entrega:".
- Usa un <input> de tipo "text".
- Haz que sea obligatorio.
- Dale un name (ej: "direccion").
- Vincula el <label> con el <input>.

(Cierra el primer fieldset)


TAREA 6: Crea un segundo <fieldset> para los detalles del pedido.
- Añade una leyenda (<legend>) que diga "Detalles del Pedido".

TAREA 7: Lista desplegable para el tamaño de la pizza.
- Usa una etiqueta <label> con el texto "Tamaño de la Pizza:".
- Usa una etiqueta <select> con un name apropiado (ej: "tamano_pizza").
- Dentro del select, añade tres <option>:
  1. Pequeña (value="pequena")
  2. Mediana (value="mediana")
  3. Grande (value="grande")
- Vincula el <label> con el <select>.


TAREA 8: Botones de radio para el tipo de masa.
- Crea dos opciones con <input type="radio">.
- Ambas deben compartir el mismo name (ej: "tipo_masa") para que solo se pueda elegir una.
- Opción 1:
  • value="fina"
  •


TAREA 9: Checkboxes para los ingredientes extra.
- Crea tres opciones con <input type="checkbox">.
- Cada una debe tener un name y value únicos.
- Opción 1: name="extra_queso", value="si" —
- Opción 2: name="extra_jamon", value="si" —
- Opción 3: name="extra_champinones", value="si" —
- Vincula cada label con su input.


TAREA 10: Campo para instrucciones especiales.
- Usa una etiqueta <label> que diga "Instrucciones Especiales:".
- Usa una etiqueta <textarea>.
- Dale un name (ej: "instrucciones").
- Añade un placeholder que diga "Ej: Sin cebolla, por favor.".
- Vincula el label con el textarea.

(Cierra el segundo fieldset)


TAREA 11: Botón de envío.
- Crea un botón que envíe el formulario.
- Puedes usar <button type="submit"> o <input type="submit">.
- El texto del botón debe ser "Realizar Pedido".

(Cierra la etiqueta del formulario)
