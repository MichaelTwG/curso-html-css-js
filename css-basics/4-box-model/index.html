Entendiendo el Box Model

El box model controla cómo se muestran y posicionan los elementos. Cada elemento es una caja con 6 partes:

  1. content: El contenido real (texto, imagen).
  2. width/height: El tamaño del contenido.
  3. padding: Relleno interno (protección).
  4. border: El borde visible.
  5. margin: Espacio externo (separación).

3. Margen (margin)

Espacio invisible fuera de la caja. Separa los elementos entre sí.

⚠️ Nota: Colapso de Márgenes Verticales

Fíjate en las cajas de arriba. Ambas tienen margin: 30px.

La lógica diría: 30px de abajo de la primera + 30px de arriba de la segunda = 60px de separación.

Realidad: Solo hay 30px. En vertical, los márgenes no se suman, se combinan (gana el mayor).

Tip de Centrado: Para centrar una caja horizontalmente usa: margin: 0 auto;

4. Borde (border)

El borde necesita 3 datos: grosor, estilo y color. Ej: border: 3px solid black;

5. Establecer el tamaño total de la caja con Box Sizing

box-sizing es la propiedad que define como se va a calcular el tamaño total de la caja, es decir si definimos que una caja tena un widht de 100px, luego le agregamos un padding de 20px (todos los lados) y un border de 3px (todos los lados), entonces el width final de la caja seria 146px en total (con el valor content-box), en el caso de que la propiedad este definida como border-box se calcularia internamente para ocupar 100px de ancho en total, ocuparia 54px de width, 20px de padding mas 3px de border (recordar que al poner padding 20px se le agrega 20px a la izquierda y 20px a la derecha, lo mismo sucede con border). Entonces 54px + 20px + 20px + 3px + 3px = 100px.
