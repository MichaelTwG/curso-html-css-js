Entendiendo el concepto de Box Model en html/css

El box model o modelo de cajas es un conjunto de reglas y propiedades que controlan como se muestran y posicionan los elementos dentro de nuestra web. Existen 6 partes basicas a la hora de trabajar con el box model de un elemento. Estas son:

  1. width para definir el tamaño horizontal de la caja
  2. height para definir el tamaño vertical de la caja
  3. contentes el contenido y la base de toda la caja puede ser, texto, imagen, video, audio, etc
  4. padding es el relleno que hay entre el contenido y el borde
  5. border es el borde de la caja
  6. margin es un espacio que separa la caja de otras cajas
