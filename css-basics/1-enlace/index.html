Enlazando CSS

El archivo CSS se enlaza mediante la etiqueta link dentro del head la etiqueta tiene 2 atributos. rel que significa el tipo de documento que va a abrirse en este caso como es un archivo CSS el valor tiene que ser stylesheet y por ultimo el atributo href que es la ruta del archivo CSS. El resultado es:

<link rel="stylesheet" href="1-enlace.css">
